# Quantum Forge 🌌

Procedural generation playground powered by **Wave Function Collapse (WFC)**. Created using **Google Antigravity** with minimal human input.

## 🚀 Live Demo
The project is hosted as a static site. Explore the generators below:
- **[Island Generation](index.html)**: Procedural terrain growth with custom weights and strategies (BFS/DFS).
- **[Floor Plan Generation](floor-plan.html)**: Organic room splatting algorithm for interior layouts.
- **[How It Was Made](about.html)**: A detailed log of the AI development process and prompt history.

## 🛠 Features
- **Wave Function Collapse**: A constraint-solving algorithm for tile-based procedural generation.
- **Organic Room Splatting**: A hybrid approach using circular splats and Minimum Spanning Trees (MST) to create natural-looking floor plans.
- **Customizable Weights**: Tweak terrain probabilities to generate unique biomes.
- **Hybrid Propagation**: Support for both Depth-First (tendril-like) and Breadth-First (bubble-like) generation strategies.
- **Responsive Design**: Premium dark-mode UI with a mobile-first glassmorphism aesthetic.

## 🏗 Setup & Local Development
Since this project has been converted to a static site, you can run it directly in any browser:
1. Clone the repository.
2. Open `index.html` in your browser.

*Note: The original project was developed using a Flask backend, but has been flattened for GitHub Pages compatibility.*
